# Music Hit Formula

Кажется, нейросети и генеративные модели стремительно меняют и музыкальную индустрию. 
Сегодня артисты используют их для обработки звука, а также создания полноценных элементов треков — вокала, инструментала, структуры песен.

Интересный пример — Kanye West, который в 2025 году представил раннюю версию альбома Bully, в значительной степени созданного с использованием ИИ. В альбом вошли, в том числе, треки на испанском языке, которым артист не владеет. Это позволило ему выйти за рамки привычного звучания и экспериментировать с формой и содержанием.

Сам артист [сравнивает ИИ с автотюном](https://www.complex.com/music/a/tracewilliamcowen/ye-ai-autotune): технологией, которую сначала не принимали, но которая со временем стала стандартом индустрии. По его мнению, ИИ так же радикально изменит процесс создания музыки. 

Отметим, что Канье был одним из первых, кто продвигал автотюн. На своем альбоме 808s & Heartbreak, выпущенном в 2008 году, артист не стеснялся использовать этот инстурмент, что позволило ему добиться совершенно нового звучания по сравнению в трилогией альбомов The сollege Dropоut - Late Registration - Graduation. 

В общем, есть все основания полагать, что ИИ может стать таким же геймченджером.

Тогда возникает гипотеза:
если понять, какие характеристики делают трек популярным, то при помощи ИИ их можно будет использовать для генерации музыки, которая будет востребована, что определенно принесет прибыль

## Цель проекта

Выявить характеристики музыкальных треков, которые связаны с их высокой популярностью на одном из самых крупных стриминге - Spotify

## Данные

Основной датасет был получен с [Kworb.net](https://kworb.net/spotify/country/global_daily_totals.html).

Датасет представляет собой агрегированную статистику по глобальным чартам Spotify и формируется на основе ежедневных позиций треков. 

### Для расширения датасета использовался [Genius API](https://docs.genius.com/).

**Использованные эндпоинты:**

- **`/search`** — Поиск трека по строке запроса (исполнитель + название).
- **`/songs/{song_id}`** — Данные о треке.
- **`/artists/{artist_id}`** — Данные об исполнителе.
- **`/artists/{artist_id}/songs`** — Список треков артиста.
- **`/referents`** — Аннотации к тексту песни.

## Пайплайн нашего решения

1. **Сбор исходного датасета.** Был спарсен датасет Kworb с агрегированными данными по популярным трекам Spotify.
2. **Обогащение данных через API Genius.** Для каждого трека выполнялся поиск в Genius, после чего последовательно запрашивались данные о песне, исполнителе, аннотациях и треках артиста. Это позволило существенно расширить исходный датасет.
3. **Формирование признаков.** На основе объединённых данных были сформированы признаки, включающие метаданные, поведенческие показатели и бинарные характеристики треков.
4. **Анализ обложек треков.** Из изображений обложек извлекались цветовые характеристики, а также рассчитывались показатели яркости и контраста, что позволило учитывать не только то, как трек звучит, но и то, как он воспринимается слушателем при зрительном контакте.
5. **Анализ данных.** Проводился исследовательский анализ данных, включая изучение распределений и корреляций между признаками и популярностью треков.
6. **Визуализация данных.** Была проведена работа по наглядному предсавлению исследумеых зависимостей.
7. **Интерпретация результатов.** На основе анализа были выделены признаки, потенциально влияющие на успех треков.
