# Формула музыкального хита

Опеределение характеристик песни, которые напрямую коррелируют с высокими позициями в рейтингах прослушиваний. 

## Команда проекта

- **Вячеслав Костин** (Атос)
- **Глеб Лавренко** (Портос)
- **Дамир Искулов** (Арамис)
